Üzərində işlənilən Kaggle dataseti - https://www.kaggle.com/datasets/jp797498e/twitter-entity-sentiment-analysis

Dataset haqqında məlumat:
Bu məlumat toplusu Twitter üzərində aparılmış entitet səviyyəsində məzmun (sentiment) analizinə aiddir. Verilən tapşırıq — bir mesaj və həmin mesajda adı keçən konkret entitet (məsələn, şəxs, təşkilat, brend və s.) əsasında, həmin entitetə qarşı olan emosional münasibəti müəyyən etməkdir. Bu məlumat toplusunda üç əsas sinif mövcuddur:

Positive (Müsbət)
Negative (Mənfi)
Neutral (Neytral)
Mesajın entitetlə əlaqəsi olmadığı (yəni əlaqəsiz və ya irrelevant) hallar da "Neutral" (Neytral) sinifinə aid edilir.
